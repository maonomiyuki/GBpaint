# GBpaint

ゲームボーイ向け（160x144 / 4色固定）静的Webペイントツールです。

## 起動
`index.html` をブラウザで開くだけで使えます。

## 主な機能
- ペン / 消しゴム / 塗りつぶし（4近傍バケツ）
- ブラシサイズ 1px / 2px / 4px
- パターン塗り（透明保持型）
  - 2x2: 25/50/75%
  - 4x4: 12.5/25/37.5/50/62.5/75/87.5%
  - 反転対応
- Undo / Redo（最大100履歴）
- JSON保存 / JSON読込（version付き）
- PNG書き出し（等倍・整数倍・正方形）
- 8x8タイルスナップ
- ユニークタイル数カウント（192制限警告）
- ユニークタイル一覧とヒートマップ
- 日本語 / 英語 切替（localStorage保持）
