# GBpaint

ゲームボーイ向け（160x144 / 4色固定）静的Webペイントツールです。

## 起動
`index.html` をブラウザで開くだけで使えます。

## 主な機能
- ペン / 消しゴム / 塗りつぶし
- ブラシサイズ 1px / 2px / 4px
- パターン塗り（透明保持型: 1px市松、2x2 25/50/75%、反転対応）
- Undo / Redo（最大100履歴）
- JSON保存 / JSON読込
- PNG書き出し（等倍、整数倍、正方形SNS向け）
- 8x8タイルスナップ
- ユニークタイル数カウント（192制限警告）
- ユニークタイル一覧とヒートマップ
